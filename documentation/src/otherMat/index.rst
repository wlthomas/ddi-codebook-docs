Other Materials
===============

.. toctree::
   :caption: Table of contents
   :maxdepth: 3
   
Other Study Related Materials `(otherMat) <https://docs.ddialliance.org/DDI-Codebook/2.5/xmlschema/schemas/codebook_xsd/elements/otherMat.html>` allows reference to external materials. This allows elements with the attribute pubRefs (IDREFS) to reference external materails through this standard structure. 

Use primarily for material types not included elsewhere in DDI-Codebook (see also):

- stdyDscr/studyDevelopment/developmentActivity/resources
- stdyDscr/method/dataColl/sources
- stdyDscr/method/dataColl/resInstu
- stdyDscr/othrStdyMat
   - relMat
   - relStdy
   - relPubl
   - othRefs
